Overview

What is Addison?

Addison is a task management app that uses AI and cognitive science-based design to help users — especially those with ADHD or executive dysfunction — break overwhelming to-do lists into manageable, focused action. Instead of a static list, Addison surfaces one task at a time and uses AI to decompose complex tasks into step-by-step subtasks with time estimates.

The app features multiple prioritization modes (Smart, Quick Wins, Deadline), a Focus Mode that runs a countdown timer on a single task, and daily nudges to keep momentum going.

Features

How Addison works

🧠

AI Task Decomposition

Drop in any task and Addison breaks it into timed subtasks using Azure OpenAI — with importance levels and duration estimates.

⏱

Focus Mode

One task at a time with a countdown timer. Reduces decision fatigue and keeps attention where it matters.

⚡

Smart Prioritization

Three modes — Smart (weighted score), Quick Wins (shortest first), Deadline — so users work the way their brain works today.

🔔

Daily Nudges

Scheduled push notifications for daily recaps and inactivity nudges to gently maintain momentum.
